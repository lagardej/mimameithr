== Naming convention

C# identifiers use the original Old Norse form with Unicode characters.
Filenames and directories use an ASCII romanization to avoid filesystem issues.

Romanization rules for filenames:

* `ð` (eth) → `th`
* `þ` (thorn) → `th`
* Accented vowels (`ö`, `í`, `á`, `ø`) are dropped or replaced by their unaccented equivalent.

Examples:

[cols="1,1"]
|===
|C# identifier |Filename

|Hlóðyn |Hlothyn/
|Iðunn |Ithunn/
|Urðr |Urth.cs
|Verðandi |Verthandi.cs
|===
