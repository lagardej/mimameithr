== Naming convention

C# identifiers use the original Old Norse form with Unicode characters.
Filenames and directories use an ASCII romanization to avoid filesystem issues.

Romanization rules:

* `ð` (eth) → `th`
* `þ` (thorn) → `th`
* `ö`, `ø` → `o`
* `á`, `à` → `a`
* `í`, `ì` → `i`
* `ú`, `ù` → `u`
* Trailing accents dropped where the character is already romanized.

Examples:

[cols="1,1"]
|===
|C# identifier |Romanized

|Biðot |Bith
|Hlóðyn |Hlothyn
|Iðunn |Ithunn
|Urðr |Urth
|Verðandi |Verthandi
|===
