= Mass Scale Reference

The `Mass` parameter uses a 1–100 scale that maps piecewise-exponentially from 10¹² kg to 10³² kg, across three zones:

[cols="<,<,<"]
|===
|Score range |Exponent range |Mass range

|1 – 40 |10¹² – 10²⁶ |10¹² kg – 10²⁶ kg
|40 – 70 |10²⁶ – 10²⁹ |10²⁶ kg – 10²⁹ kg
|70 – 100 |10²⁹ – 10³² |10²⁹ kg – 10³² kg
|===

Score is linearly interpolated against exponent within each zone, then mass = 10^exponent.
Implemented by `Range100.PiecewiseExponentialScale(mass, exponents: [12, 26, 29, 32])` (default thresholds `[40, 70, 100]`).
