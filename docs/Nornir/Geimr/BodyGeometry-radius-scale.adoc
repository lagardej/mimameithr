= Radius Scale Reference

The `Radius` parameter uses a 1–100 scale that maps exponentially from 1 km to 10⁹ km.
`1 + 11 * log10(radius in km)` is the formula used to convert from km to the scale.
The following table provides a reference for common celestial bodies and their corresponding radius scale values.
