= Radius Scale Reference

The `Radius` parameter uses a 1–100 scale that maps piecewise-exponentially from 1 km to 10⁹ km, across three zones:

[cols="<,<,<"]
|===
|Score range |Exponent range |Radius range

|1 – 40 |10⁰ – 10⁴ |1 km – 10,000 km
|40 – 70 |10⁴ – 10⁶ |10,000 km – 1,000,000 km
|70 – 100 |10⁶ – 10⁹ |1,000,000 km – 1,000,000,000 km
|===

Score is linearly interpolated against exponent within each zone, then radius = 10^exponent.
Implemented by `Range100.PiecewiseExponentialScale(radius, exponents: [0, 4, 6, 9])` (default thresholds `[40, 70, 100]`).
