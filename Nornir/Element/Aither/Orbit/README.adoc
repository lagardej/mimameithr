= Orbit

Keplerian orbital state of a planetary body around its parent star.

== Parameters

[cols="1,1,3",options="header"]
|===
| Name | Unit | Purpose
| SemiMajorAxis | m | Semi-major axis of the orbit.
| Eccentricity | - | Shape of the orbital ellipse. 0 = circular, approaching 1 = parabolic limit.
| OrbitalPeriod | s | Sidereal orbital period.
| InitialMeanAnomaly | ° | Mean anomaly at epoch (t = 0).
|===

== States

[cols="1,1,3",options="header"]
|===
| Name | Unit | Purpose
| MeanAnomaly | ° | Current mean anomaly.
| OrbitalAngle | ° | Current true anomaly.
| DistanceFromStar | m | Current distance between the body and its parent star.
|===
