Asthenospheric heat flux from body mass, radius, and age.

== Model

Provides physics calculations for planetary mantle properties.

== API

[cols="1,2,2,3",options="header"]
|===
| Method | Parameters | Returns | Description
| AsthenosphericHeatFlux | Duration age, Mass mass, Length radius | HeatFlux | Derives asthenospheric heat flux from body mass, radius, and age. Uses specific radiogenic heat production scaled by mass, decayed by age, and normalized by surface area. No Earth reference values.
|===
