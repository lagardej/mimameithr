# Kanban

Folders are workflow. Move task file between folders; file itself holds task notes.

## Folders

| Status    | Description    |
|-----------|----------------|
| 0_Backlog | not started    |
| 1_Todo    | ready to start |
| 2_Doing   | in progress    |
| 3_Done    | finished       |
| Archive   | parked         |
| Rejected  | discarded      |
